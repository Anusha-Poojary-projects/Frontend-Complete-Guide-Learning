What are Operators?

Operators are symbols that perform operations on operands (values and variables). They are the building blocks of expressions in JavaScript.

Operator Anatomy

// Basic expression let result = 10 + 5; // Breakdown: // 10 and 5 are operands // + is the operator // = is the assignment operator // result is the variable storing the outcome

Unary Operators

Operate on a single operand

let x = -5; let isActive = !true;

Binary Operators

Operate on two operands

let sum = 10 + 5; let isGreater = 10 > 5;

Ternary Operator

Operates on three operands

let message = age >= 18 ? "Adult" : "Minor";

Arithmetic Operators

Arithmetic operators perform mathematical operations on numeric operands.

Operator Description Example Result + Addition 10 + 5 15 - Subtraction 10 - 5 5 * Multiplication 10 * 5 50 / Division 10 / 5 2 % Modulus (Remainder) 10 % 3 1 ** Exponentiation 2 ** 3 8

Special Cases

// String concatenation with + let fullName = "John" + " " + "Doe"; // "John Doe" // Unary plus converts to number let num = +"42"; // 42 // Increment/Decrement operators let count = 5; count++; // 6 (post-increment) ++count; // 7 (pre-increment)

Comparison Operators

Comparison operators compare two values and return a boolean result (true or false).

Operator Description Example Result == Equal (loose) 5 == "5" true === Equal (strict) 5 === "5" false != Not equal (loose) 5 != "6" true !== Not equal (strict) 5 !== "5" true > Greater than 10 > 5 true < Less than 10 < 5 false >= Greater than or equal 10 >= 10 true <= Less than or equal 10 <= 5 false

Loose vs Strict Equality

// Loose equality (==) performs type coercion 5 == "5" // true true == 1 // true null == undefined // true // Strict equality (===) checks both value and type 5 === "5" // false true === 1 // false null === undefined // false // Best practice: Always use === and !==

Logical Operators

Logical operators are used with boolean values and return boolean values, but they can work with any type.

Operator Description Example Result && Logical AND true && false false || Logical OR true || false true ! Logical NOT !true false

Short-Circuit Evaluation

// && returns the first falsy value or the last value let result1 = true && "hello" && 42; // 42 let result2 = true && "" && 42; // "" // || returns the first truthy value or the last value let result3 = false || "hello" || 42; // "hello" let result4 = false || 0 || null; // null

Other Important Operators

JavaScript includes several other operators that are essential for advanced programming.

Conditional (Ternary) Operator

// Syntax: condition ? exprIfTrue : exprIfFalse let age = 20; let status = (age >= 18) ? "adult" : "minor"; console.log(status); // "adult"

Type Operators

// typeof operator typeof 42; // "number" typeof "hello"; // "string" typeof true; // "boolean" typeof {}; // "object" typeof null; // "object" (bug in JS) // instanceof operator let arr = [1, 2, 3]; arr instanceof Array; // true

Bitwise Operators

// AND: & 5 & 3; // 1 (binary: 101 & 011 = 001) // OR: | 5 | 3; // 7 (binary: 101 | 011 = 111) // XOR: ^ 5 ^ 3; // 6 (binary: 101 ^ 011 = 110) // NOT: ~ ~5; // -6 (inverts all bits)

Nullish Coalescing Operator

// ?? returns the right operand when the left is null or undefined let name = null; let displayName = name ?? "Guest"; console.log(displayName); // "Guest" // Different from || which checks for any falsy value let count = 0; let actualCount = count || 10; // 10 (0 is falsy) let actualCount2 = count ?? 10; // 0 (0 is not null/undefined)

Operator Precedence

Operator precedence determines the order in which operators are evaluated in an expression.

Precedence Operator Description Associativity 21 () Grouping n/a 20 ++, -- Increment/Decrement n/a 17 ** Exponentiation right-to-left 16 *, /, % Multiplication/Division left-to-right 15 +, - Addition/Subtraction left-to-right 14 <<, >>, >>> Bitwise shift left-to-right 11 <, <=, >, >= Relational left-to-right 10 ==, ===, !=, !== Equality left-to-right 6 && Logical AND left-to-right 5 || Logical OR left-to-right 3 =, +=, -=, etc. Assignment right-to-left

Examples

// Without parentheses let result1 = 3 + 4 * 2; // 11 (multiplication first) // With parentheses for clarity let result2 = (3 + 4) * 2; // 14 // Mixed operators let result3 = true || false && false; // true (&& has higher precedence) // Assignment is right-associative let a, b, c; a = b = c = 5; // All variables get 5

JavaScript Operators: Key Takeaways

Operator Types

  • Arithmetic: +, -, *, /, %, **
  • Assignment: =, +=, -=, *=, etc.
  • Comparison: ==, ===, !=, !==, >, <, >=, <=
  • Logical: &&, ||, !
  • Special: ternary, typeof, instanceof, ??

Best Practices

  • Always use === and !== for comparisons
  • Use parentheses to make precedence explicit
  • Understand short-circuit evaluation
  • Prefer nullish coalescing (??) over logical OR (||) when appropriate
  • Be aware of type coercion in loose comparisons

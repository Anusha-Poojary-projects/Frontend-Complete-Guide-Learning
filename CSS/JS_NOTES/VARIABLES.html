What are Variables?

Variables are containers for storing data values. In JavaScript, variables are used to hold information that can be referenced and manipulated throughout your code.

Key Characteristics:

Named Containers
Each variable has a unique name (identifier)
Mutable Values
Values can be changed after declaration (except const)
Dynamic Typing
Variable types are determined at runtime
Memory Management
JavaScript handles memory allocation automatically

Variable Declaration

JavaScript provides three ways to declare variables:

var
Function-scoped - Can be redeclared and updated
Older way, avoid in modern JavaScript
let
Block-scoped - Can be updated but not redeclared
Use for variables that will change
const
Block-scoped - Cannot be redeclared or reassigned
Use for variables that won't change (preferred by default)

Examples:

// var (avoid) var name = "John"; var name = "Jane"; // This works but is bad practice // let (for changing values) let age = 25; age = 26; // This works // let age = 27; // This would cause an error // const (for constant values) const PI = 3.14159; // PI = 3.14; // This would cause an error

JavaScript Data Types

JavaScript has 8 data types, categorized into two groups:

Primitive Types (7)

string - Text data
number - Numeric values
bigint - Large integers
boolean - true/false values
undefined - Variable declared but not assigned
null - Intentional absence of value
symbol - Unique identifiers

Reference Type (1)

object - Collections of data
Includes: Arrays, Functions, Dates, etc.

Dynamic Typing Example:

let value = "Hello"; // string value = 42; // number value = true; // boolean value = [1, 2, 3]; // object (array) value = null; // null

JavaScript automatically determines the type based on the assigned value!

Primitive Data Types

Primitive types are immutable and stored directly in memory.

string

let name = "Alice"; let greeting = 'Hello, ' + name; let message = `Welcome, ${name}!`;

number

let integer = 42; let float = 3.14; let negative = -10; let infinity = Infinity; let notANumber = NaN;

boolean

let isActive = true; let isComplete = false; let hasPermission = Boolean(1); // true

null & undefined

let empty = null; // intentional empty value let uninitialized; // undefined (not assigned) let alsoUndefined = undefined;

Objects and Reference Types

Objects store collections of data and are stored by reference.

Object

let person = { name: "John", age: 30, isActive: true }; let car = new Object(); car.brand = "Toyota";

Array

let colors = ["red", "green", "blue"]; let numbers = new Array(1, 2, 3, 4, 5); console.log(colors[0]); // "red" console.log(numbers.length); // 5

Function

function greet(name) { return "Hello, " + name; } let sayHello = function(name) { return "Hi, " + name; }; let arrowFunc = (name) => `Hey, ${name}!`;

Type Checking and Conversion

Understanding and working with different data types.

Type Checking

typeof "hello" // "string" typeof 42 // "number" typeof true // "boolean" typeof undefined // "undefined" typeof null // "object" (bug in JavaScript!) typeof {} // "object" typeof [] // "object" typeof function(){} // "function" // Better way to check arrays Array.isArray([1,2,3]) // true // Check for null specifically value === null // true if null

Type Conversion

// String conversion String(42) // "42" (42).toString() // "42" // Number conversion Number("42") // 42 parseInt("42px") // 42 parseFloat("3.14") // 3.14 // Boolean conversion Boolean(0) // false Boolean("hello") // true !!"hello" // true (double negation)

Truthy and Falsy Values

In JavaScript, values are either truthy or falsy when evaluated in a boolean context.

Truthy Values

These evaluate to true:

  • Any non-empty string: "hello"
  • Any number except 0: 42, -1, 3.14
  • Any object: {}, [], function(){}
  • The string "0"
  • The string "false"

Falsy Values

These evaluate to false:

  • false
  • 0
  • -0
  • 0n (BigInt zero)
  • "" (empty string)
  • null
  • undefined
  • NaN

Practical Example:

if (userInput) { // Runs if userInput is truthy console.log("Valid input!"); } else { // Runs if userInput is falsy console.log("Please enter something!"); }

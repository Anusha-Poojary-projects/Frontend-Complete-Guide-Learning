What are Pseudo-Classes?

Pseudo-classes are keywords added to selectors that specify a special state of the selected element.

Key Characteristics:

State-Based
Apply styles based on user interaction or element state
Dynamic
Styles change without modifying HTML
Positional
Target elements based on their position in the document
Form Validation
Style form elements based on validation state
/* Syntax */ selector:pseudo-class { property: value; } /* Example */ a:hover { color: red; }

What are Pseudo-Elements?

Pseudo-elements are used to style specific parts of an element. They create "phantom" elements that don't exist in the HTML.

Key Characteristics:

Content Insertion
Add decorative content before/after elements
Styling Parts
Style specific parts like first letter or line
Selection Styling
Customize how selected text appears
Counters
Create automatic numbering systems
/* Syntax */ selector::pseudo-element { property: value; } /* Example */ p::first-letter { font-size: 2em; }

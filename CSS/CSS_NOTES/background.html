What are Background Images?

CSS Background properties allow you to add decorative images, gradients, and patterns to HTML elements without affecting the document structure. They provide powerful control over how backgrounds are displayed and positioned.

Key Benefits:

Non-Intrusive
Doesn't affect HTML structure or SEO
Flexible Styling
Full control over size, position, and repetition
Performance
Can be optimized and cached separately
Creative Freedom
Combine images, gradients, and patterns

Background Properties

CSS provides several properties to control background appearance:

Individual Properties

background-image
Sets the background image(s)
background-size
Controls image sizing
background-position
Sets image position
background-repeat
Controls image repetition
background-attachment
Sets scroll behavior
background-clip
Defines painting area

Shorthand Property

/* Shorthand syntax */ background: [attachment] [position] / [size] [repeat] [origin] [clip] [image]; /* Example */ background: fixed center/cover no-repeat url('image.jpg');

Background Size

The background-size property controls how background images are sized.

cover

Cover - Fills container
background-size: cover;

contain

Contain - Fits entirely
background-size: contain;

100% 100%

Stretch - Distorts image
background-size: 100% 100%;

auto (default)

Auto - Original size
background-size: auto;

Background Position

The background-position property controls where the background image is placed.

center (default)

Centered

top left

Top Left

bottom right

Bottom Right

50% 25%

Custom Position
/* Keyword values */ background-position: center; background-position: top left; background-position: bottom right; /* Percentage values */ background-position: 50% 25%; /* Length values */ background-position: 100px 50px;

Background Repeat

The background-repeat property controls how background images are repeated.

repeat (default)

Repeated

no-repeat

Single Image

repeat-x

Horizontal Only

repeat-y

Vertical Only
background-repeat: repeat; /* Default */ background-repeat: no-repeat; /* Single image */ background-repeat: repeat-x; /* Horizontal only */ background-repeat: repeat-y; /* Vertical only */ background-repeat: space; /* Spaced repetition */ background-repeat: round; /* Scaled repetition */

Advanced Background Features

Modern CSS provides powerful advanced background capabilities.

Multiple Backgrounds

Multiple Backgrounds
background-image: linear-gradient(45deg, rgba(255,0,0,0.5), rgba(0,255,0,0.5)), url('pattern.png'); background-size: cover, 50px 50px; background-repeat: no-repeat, repeat;

Background Attachment

Fixed Background (Parallax Effect)
background-attachment: fixed; /* Parallax effect */ background-attachment: scroll; /* Default behavior */

Background Clip

Text with Background Clip
background: linear-gradient(45deg, #667eea, #764ba2); -webkit-background-clip: text; -webkit-text-fill-color: transparent; background-clip: text;

Practical Examples

1. Hero Section

Welcome to Our Website

Beautiful background with perfect text overlay

.hero { background-image: url('hero.jpg'); background-size: cover; background-position: center; background-repeat: no-repeat; min-height: 500px; display: flex; align-items: center; justify-content: center; color: white; }

2. Gradient Backgrounds

Linear Gradient
Radial Gradient
.linear-bg { background: linear-gradient(45deg, #ff6b6b, #4ecdc4); } .radial-bg { background: radial-gradient(circle, #667eea, #764ba2); }

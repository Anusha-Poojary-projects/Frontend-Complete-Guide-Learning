What are CSS Transformations?

CSS Transformations allow you to rotate, scale, skew, or translate elements without affecting the normal document flow.

Key Benefits:

Visual Effects
Create 2D and 3D visual effects
Performance
Hardware-accelerated for smooth performance
Non-Disruptive
Don't affect layout or other elements
Versatile
Combine multiple transformations
/* Syntax */ transform: function(value); /* Example */ .element { transform: rotate(45deg) scale(1.2); }

Transformation Types

Here are the main transformation functions:

Rotate

Rotate
transform: rotate(45deg);

Scale

Scale
transform: scale(1.5);

Translate

Translate
transform: translate(50px, 30px);

Skew

Skew
transform: skew(20deg, 10deg);

3D Transformations

Transformations can also be applied in 3D space using functions like rotateX(), rotateY(), rotateZ(), and translate3d().

What are CSS Animations?

CSS Animations allow you to change property values smoothly over time, creating motion effects without JavaScript.

Key Components:

@keyframes
Define the animation sequence
Animation Properties
Control timing, duration, and behavior
Performance
Optimized for smooth rendering
Versatility
Animate any animatable CSS property
/* Define animation */ @keyframes slideIn { from { transform: translateX(-100px); } to { transform: translateX(0); } } /* Apply animation */ .element { animation: slideIn 1s ease-in-out; }

Animation Properties

Control your animations with these properties:

animation-name

Name of the @keyframes animation

animation-duration

How long the animation takes to complete

animation-timing-function

Speed curve of the animation (ease, linear, etc.)

animation-delay

Delay before the animation starts

animation-iteration-count

Number of times to repeat (infinite, 3, etc.)

animation-direction

Direction of animation (normal, reverse, alternate)

animation-fill-mode

Styles applied before/after animation (forwards, backwards)

animation-play-state

Pause or run animation (paused, running)

/* Shorthand */ animation: name duration timing-function delay iteration-count direction fill-mode;

Transitions vs Animations

Transitions

  • Define start and end states
  • Triggered by state changes (hover, focus)
  • Simple property changes
  • Cannot loop or have complex timing
  • Uses transition property
Hover Me!
.element { transition: all 0.5s ease; } .element:hover { transform: rotate(360deg); }

Animations

  • Define multiple keyframes
  • Run automatically or on trigger
  • Complex sequences possible
  • Can loop, reverse, and pause
  • Uses @keyframes and animation
Bouncing!
@keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-30px); } } .element { animation: bounce 2s infinite; }

Animations & Transformations Cheat Sheet

Transform Functions

/* 2D Transforms */ transform: rotate(angle); transform: scale(x, y); transform: translate(x, y); transform: skew(x-angle, y-angle); transform: matrix(n,n,n,n,n,n); /* 3D Transforms */ transform: rotateX(angle); transform: rotateY(angle); transform: rotateZ(angle); transform: translate3d(x, y, z); transform: scale3d(x, y, z); transform: perspective(n);

Animation Properties

/* Keyframes */ @keyframes name { from { /* styles */ } to { /* styles */ } } /* Animation Properties */ animation-name: name; animation-duration: 2s; animation-timing-function: ease; animation-delay: 0.5s; animation-iteration-count: infinite; animation-direction: alternate; animation-fill-mode: forwards; animation-play-state: running; /* Shorthand */ animation: name 2s ease 0.5s infinite alternate forwards;

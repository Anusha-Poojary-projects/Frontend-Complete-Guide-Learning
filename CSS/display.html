CSS Positioning Example

This example demonstrates different CSS positioning methods: static, relative, absolute, fixed, and sticky.

Static Positioning

This box uses the default static positioning. It flows naturally with the document layout.

Relative Positioning

This box is positioned relative to its normal position. The top and left properties move it 50px down and 50px right from where it would normally be.

Absolute Positioning

This box is positioned absolutely within the relative box. It is taken out of the normal document flow and positioned at (20px, 20px) from the top-left corner of the relative box.

Sticky Positioning

This box will stick to the top of the viewport when you scroll past it, but only within the bounds of its parent container.

Fixed Positioning

This box is fixed to the viewport. It stays in the same position even when you scroll the page.

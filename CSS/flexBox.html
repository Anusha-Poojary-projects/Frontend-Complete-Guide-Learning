CSS Flexbox

What You Will Learn

This guide helps you understand CSS Flexbox and lets you practice layout alignment interactively.

Steps for Students

  1. Observe Flex Container: Notice the container has display: flex and how its items behave.
  2. Try Changing Flex Direction: Experiment with flex-direction: row | column | row-reverse | column-reverse.
  3. Justify Content: Change justify-content values (flex-start, flex-end, center, space-between, space-around, space-evenly) and see how items align along the main axis.
  4. Align Items: Modify align-items (stretch, flex-start, flex-end, center, baseline) to align items along the cross axis.
  5. Align Self: Apply align-self on a single item to override align-items.
  6. Flex Grow, Shrink, Basis: Play with flex: 1, flex: 2, flex: 0.5 to see how items expand or shrink.
  7. Practice: Create multiple flex containers and experiment with combinations of direction, alignment, and spacing.
  8. Reflect: Think about how Flexbox is used in real-world layouts like navbars, cards, and galleries.

Tips for Learning

  • Use DevTools to change flex properties live and observe behavior instantly.
  • Take notes on how each property affects main and cross axes.
  • Experiment with nested flex containers for complex layouts.
  • Try using Flexbox in mini-projects to solidify your understanding.

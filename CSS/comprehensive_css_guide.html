A Beginner's Guide to CSS (with Side-by-Side Examples)

What is CSS?

CSS (Cascading Style Sheets) is the language for styling webpages. It works with HTML to control the visual presentation of your content, including layout, colors, and fonts.

CSS Selectors: How to Target Elements

Selectors are patterns used to select the HTML element(s) you want to style.

1. Element Selector

Targets all elements of a specific type, like all paragraphs (`

`).

This is a standard paragraph, styled by the `p` element selector.

/* This CSS rule targets all 'p' elements */
p {
    color: #555;
}

2. Class Selector

Targets all elements with a specific `class` attribute. Starts with a period (`.`).

This paragraph has a class of "highlight".

This div also has the "highlight" class.
<!-- HTML -->
<p class="highlight">...</p>
<div class="highlight">...</div>

<!-- CSS -->
.highlight {
    background-color: yellow;
    font-weight: bold;
}

3. ID Selector

Targets one unique element with a specific `id` attribute. Starts with a hash (`#`).

This div has a unique ID, so it gets a special border.
<!-- HTML -->
<div id="unique-element">...</div>

<!-- CSS -->
#unique-element {
    border: 2px solid #d9534f;
    padding: 10px;
}

Common CSS Properties

4. Color and Background Color

This div has white text on a green background. 
.color-example {
    color: white;
    background-color: #5cb85c;
    padding: 15px;
}

5. Font Properties

This paragraph has a different font, size, and weight.

.font-example {
    font-size: 22px;
    font-family: 'Georgia', serif;
    font-weight: normal;
}

6. Text Alignment

This text is centered. 
.text-align-example {
    text-align: center;
}

The Box Model: Margin, Padding, and Border

Every HTML element is a box. The box model describes the space around and inside an element.

This box demonstrates Margin, Border, and Padding. 
.box-model-example {
    margin: 30px 0; /* Space outside */
    border: 5px solid #333;
    padding: 20px; /* Space inside */
    background-color: #f0ad4e;
}

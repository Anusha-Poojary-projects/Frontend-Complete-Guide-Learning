Media Queries in CSS

1. Introduction to Media Queries

Media queries are a fundamental part of responsive web design. They allow you to apply different styles to your website based on the characteristics of the device or screen size being used to view it. This ensures that your website looks great and functions well on all devices, from small mobile phones to large desktop monitors.

Here is a basic example of a media query:


    @media only screen and (max-width: 600px) {
        body {
            background-color: lightblue;
        }
    }

2. Examples of Media Queries for Different Screen Sizes

Here are some common breakpoints used in media queries:

  • Extra small (phones): ≤ 375px (e.g., 320, 360, 375)
  • Small (large phones / small phablets): 376px — 480px
  • Medium (small tablets / large phones): 481px — 767px
  • Large (tablets / small laptops): 768px — 1023px
  • Extra large (desktop): 1024px — 1279px
  • 2XL (large desktop / widescreen): 1280px — 1919px
  • 4K / very large: ≥ 1920px

For each of these screen sizes, you can write specific media queries to adjust the layout, font sizes, images, and other elements to enhance the user experience.


    @media only screen and (max-width: 375px) {
        body {
            background-color: lightcoral;
        }
    }

    @media only screen and (min-width: 376px) and (max-width: 480px) {
        body {
            background-color: lightseagreen;
        }
    }

    @media only screen and (min-width: 481px) and (max-width: 767px) {
        body {
            background-color: lightpink;
        }
    }

    @media only screen and (min-width: 768px) and (max-width: 1023px) {
        body {
            background-color: lightgoldenrodyellow;
        }
    }

    @media only screen and (min-width: 1024px) and (max-width: 1279px) {
        body {
            background-color: lightgray;
        }
    }

    @media only screen and (min-width: 1280px) and (max-width: 1919px) {
        body {
            background-color: lightcyan;
        }
    }

    @media only screen and (min-width: 1920px) {
        body {
            background-color: lightyellow;
        }
    }

3. Best Practices for Using Media Queries in Responsive Design

  • Mobile-First Approach: Start by designing for the smallest screen size and then use media queries to enhance the design for larger screens.
  • Use Relative Units: Use relative units like percentages, ems, and rems instead of fixed units like pixels to ensure better scalability across different devices.
  • Avoid Overlapping Styles: Keep your media queries organized and avoid overlapping styles that can lead to confusion and maintenance issues.
  • Test on Real Devices: Always test your designs on real devices to ensure they look and function as expected.
  • Keep It Simple: Avoid using too many breakpoints. Focus on the most important ones that will cover the majority of your users.

HTML Attributes

What are attributes?

Attributes provide additional information about HTML elements.

<tag attribute="value">content</tag>

Always written inside the opening tag
Format: name="value"
Always use quotes around values

Common Attributes

href (for links)

<a href="https://google.com">Google</a>

src (for images)

<img src="photo.jpg" alt="My photo">

id & class

<div id="header" class="blue">Content</div>

Semantic HTML

What is Semantic HTML?

Semantic HTML uses tags that describe the meaning of content, not just how it looks.

‚ùå Non-semantic

<div><div><div>

Doesn't tell us anything about content

‚úÖ Semantic

Clear meaning and purpose

Semantic Tags

<header> Top of page/section
<nav> Navigation links
<main> Main content
<article> Standalone content
<section> Thematic grouping
<footer> Bottom of page/section

🌟 What is Tailwind CSS?

Tailwind is a utility-first CSS framework. Instead of writing CSS like .header { color: blue; }, you add small classes directly in HTML: 


  Hello Tailwind!

✅ No context switching — style right in your HTML
✅ Consistent design — predefined scales for spacing, colors, etc.
✅ Responsive & interactive — built-in hover, focus, breakpoints
✅ Highly customizable — extend or replace anything

🧠 Think of it like LEGO blocks:

Each class is a small piece. Combine them to build anything!

text-xl font-bold p-4 bg-blue-100 rounded

⚙️ Setup (For Real Projects)

For learning, we’re using CDN. But for real apps, install via npm:

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init

Then configure tailwind.config.js and include Tailwind in your CSS:

@tailwind base;
@tailwind components;
@tailwind utilities;
⚠️ Note: CDN is for learning only. It’s larger and doesn’t support customization or tree-shaking.

💡 Pro Tips:

  • Combine classes: bg-blue-500 hover:bg-blue-700

  • Add transitions: transition duration-300 ease-in-out
  • Try animations: animate-pulse animate-bounce

📌 Positioning — Absolute, Relative, Fixed

Position Classes:

🧠 Positioning Explained:

  • relative — creates a positioning context for absolute children
  • absolute — positions relative to nearest positioned ancestor
  • fixed — positions relative to viewport (stays on scroll)
  • inset-0 = top-0 right-0 bottom-0 left-0 (fills container)
  • Use z-* to control stacking order

🎯 Responsive Design Strategy:

  • Mobile-first: Start with base classes, then add md:, lg: for larger screens
  • Hide/show: hidden md:block or block lg:hidden
  • Change layout: flex-col sm:flex-row
  • Adjust spacing: p-2 md:p-4 lg:p-8
